MRTK学习（一）

一、混合现实工具包（MRTK）

简介：
  • 混合现实工具包 (MRTK) 是用于构建虚拟现实 (VR) 和增强现实 (AR) 的混合现实体验的跨平台工具包。能够凭此创建一种混合现实体验，在其中用户能够移动、旋转和缩放全息对象。

二、为 Windows Mixed Reality 配置 Unity

  • 首先在Unity Hub中创建3D Core项目，并取名为”MRTK Tutorial“。

  • 切换生成平台，切换成”Universal Windows Platform“。

三、导入和配置资源

  • 导入 MRTK Unity 基础包。从 Microsoft 下载中心下载混合现实功能工具的最新版本。

四、完成为Unity配置项目

五、交互模型

​ 有三种主要的交互模型适合大多数混合现实体验。

  • 手部和运动控制器模型

    手部和运动控制器模型要求用户使用一只或两只手与全息影像交互。

    三种手部和运动控制器的形态：

  1. 用手直接操作
  2. 用手指向并提交
  3. 运动控制器

  • 免手动模型

​ 免手动模型使用户无需用手即可以与全息内容进行交互。 但是可以使用语音输入或“凝视并停留”。

  • 凝视并提交

​ 在与无法触及的全息内容交互时，最好使用“凝视并提交”。 用户凝视一个对象或 UI 元素，然后使用辅助输入单击它（“提交”到它）。 提交方法包括语音命令、按下按钮或手势。

更改场景中的视图：

  • 要向前/向左/向后/向右移动摄像机，按 W/A/S/D 键。
  • 要垂直移动相机，按 Q 和 E 键。
  • 要旋转相机，按鼠标右键，然后拖动。

持久的手：

  • 要在不继续按住某个键的情况下启用手并使它在屏幕上，请按 T（左手）或 Y（右手）。 再次按这些键可移除手。
