二、使用集合组织和布局对象

​ 在“集合”中排列对象是混合现实工具包提供的另一项实用功能。 对象集合是一个脚本，用于帮助以预定义的 3 维形状为对象数组布局。 它支持各种表面样式，包括平面、圆柱体、球体和径向。 因为它支持 Unity 中的任何对象，所以它可以用来对 2D 和 3D 对象进行布局。

1.网格对象集合

​ 可以在一组子对象上使用此类型的集合，以使用一些布局参数来组织它们。 网格对象收集可帮助快速创建控制面板或一组 Prefab 或对象。

2.散点对象集合

​ 此类型的集合只是在半径内随机布局的一组子对象。

3.图块网格对象集合

​ 可以利用这种类型的集合在基本网格布局中放置大小一致的游戏对象。

三、使用求解器创建智能对象放置

1.解释：

​ 如果最重要的选项出现在周围的菜单中，而不是固定在某个位置。混合现实工具包提供了此功能，称为求解器。 求解器使用预定义的算法计算对象的位置和方向。 求解器提供了各种行为，用于将对象附加到其他对象或系统。 例如，求解器可以是每次移动时显示的跟随对象（基于相机）。

​ 方向指示器是另一种类型的求解器。 如果想帮助跟随组件将其定向到空间中对象的所需方向，则可以使用该求解器。

​ 另一个需要了解的是点击以放置组件，它可以帮助将对象放置在表面上。 如果要将对象放置在空间网格上，此组件非常有用。

​ MRTK 的求解器位于 MRTK SDK 文件夹中。

2.练习使用方向指示器求解器

1.添加的组件：
  • DirectionalIndicator：确定作为方向指示器的对象的位置和方向。 从 SolverHandler 跟踪目标的参照点来看，此指示器将面向提供的 DirectionalTarget。
2.进入Game模式，按住鼠标右键旋转方向，当视线从RoverExplorer对象移开时，Indicator对象将出现并指向RoverExplorer对象。

3.练习使用点击以放置求解器

1.点击以放置

​ “点击以放置”是一个远距离交互组件，用于将游戏对象放置在表面上。 此组件适用于将对象放置在空间网格上。 “点击以放置”使用两个单击和头部运动的组合来放置对象。 一个单击用于开始放置，头部运动用于控制对象的位置，另一个单击用于将对象放置在场景中。

2.对RoverAssembly进行相关配置

3.进入Game模式。

​ 利用鼠标右键将视线对准”RoverAssembly”后，单击鼠标左键，然后继续点击鼠标右键继续移动视线，找到自己满意的位置后，再次单击鼠标左键进行放置，至此完成。
